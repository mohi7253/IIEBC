Online Help:

This IEBC Preliminary Map Viewer application provides the general public and other interested parties the ability to view proposed poll boundaries.

Using the Application

The IEBC Preliminary Map Viewer application allows users to view proposed Poll Boundaries. Type the name of the area of interest in the search field and press/click on the button that describes the area you are looking for e.g. if you are looking for a county named Nairobi, type in 'Nairobi' then click on the button named 'County'. A list of constituencies under the said county will appear in a grid below the search field. If you then click on one of the constituencies, the application will display a popup window on the constituency with its details.

Finding an Area of Interest

The primary way to identify an area of interest is to search for a poll area by name.

Map Based Navigation

There are several ways to interact with the map:
1. Hold down the shift key and drag a box on the map to zoom in
2. Hold Down Ctrl+Shift to zoom out
3. Roll your mouse roller to zoom in/out
4. Drag the zoom slider to zoom in/out

Searching

The search box can be used to find the area you're interested in:

1. Enter the name of the poll area you're interested in
3. Click on the item in the list to locate the area or click on the area in the list

You can enter a partial area name and then identify the correct value in the list.

Browser Compatibility

The IEBC Preliminary Map Viewer application is supported on the following browsers: Microsoft IE 8/9, Mozilla Firefox 3.6/6, Google Chrome 13 and Apple Sarari 5.1.
